# 华为用户池模拟运行说明

## 1. 环境准备

### 1.1 Python 环境

要求 Python 版本 **3.10 或 3.11**（不支持 3.12 及以上）。

### 1.2 安装 Python 依赖

在项目根目录下执行以下任一命令：

**方式一：使用 Poetry（推荐）**

```bash
pip install poetry
poetry install
```

**方式二：使用 pip**

```bash
pip install camel-ai==0.2.70 pandas==2.2.2 igraph==0.11.6 cairocffi==1.7.1 \
    sentence-transformers==3.0.0 requests_oauthlib==2.0.0
```

### 1.3 配置 API Key

模拟运行使用阿里云百炼平台的 **Qwen** 模型。请打开以下文件，将其中的 `api_key` 替换为您自己的密钥：

```
examples/weibo_simulation_qwen_huawei.py
```

找到如下代码段并修改：

```python
model_platform=ModelPlatformType.QWEN,
model_type="qwen-plus",
url="https://dashscope.aliyuncs.com/compatible-mode/v1",
api_key="<替换为您的阿里云百炼 API Key>",
```

---

## 2. 输入数据

- **用户池文件路径**：`weibo_test/top100_users_complete_data_post_followers.json`
- **格式说明**：JSON 数组，每个元素代表一位微博用户，包含用户基本信息、近期发帖内容分析、粉丝/关注数据等字段。

---

## 3. 运行模拟

在项目根目录下执行：

```bash
python examples/weibo_simulation_qwen_huawei.py
```

运行过程中日志将实时输出至终端。模拟完成后结果自动写入数据库文件。

---

## 4. 输出结果

- **数据库文件路径**：`weibo_test/weibo_sim_qwen_huawei.db`（SQLite 格式）
- 数据库中记录了模拟过程中每位智能体的发帖、转发、评论、点赞等社交行为及传播关系。

---

## 5. 查看结果

### 方式一：静态可视化（快速）

在项目根目录下执行：

```bash
python transfer_tree_vis/vis.py
```

脚本将在 `transfer_tree_vis/` 目录下生成两个 HTML 文件，直接用浏览器打开即可：

| 文件 | 说明 |
|------|------|
| `transfer_tree_vis/tree.html` | 用户传播树（静态） |
| `transfer_tree_vis/tree_dynamic.html` | 用户传播树动态传播过程 |

---

### 方式二：交互式可视化（前后端）

#### 第一步：安装前端依赖

需要提前安装 **Node.js**（建议 v18 及以上）。

```bash
cd visualization_system/frontend
npm install
```

#### 第二步：安装后端依赖

```bash
pip install fastapi==0.109.0 uvicorn==0.27.0 "pydantic>=2.12.0" \
    "pydantic-settings>=2.0.0" python-multipart==0.0.6 "pandas>=2.0.0"
```

#### 第三步：配置数据库路径环境变量

将 `weibo_sim_qwen_huawei.db` 的**绝对路径**设置为环境变量 `OASIS_DB_PATH`：

**Windows PowerShell：**

```powershell
$env:OASIS_DB_PATH = "C:\<项目绝对路径>\weibo_test\weibo_sim_qwen_huawei.db"
```

**Linux / macOS / Git Bash：**

```bash
export OASIS_DB_PATH="/path/to/project/weibo_test/weibo_sim_qwen_huawei.db"
```

#### 第四步：启动后端服务

```bash
cd visualization_system/backend
python start.py
```

后端将在 `http://localhost:8001` 启动。

#### 第五步：启动前端服务

```bash
cd visualization_system/frontend
npm run dev
```

前端将在 `http://localhost:5173` 启动。

#### 第六步：访问页面

在浏览器中打开：

```
http://localhost:5173
```
